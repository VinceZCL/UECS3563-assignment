# UECS3563 Assignment

## Features
* PostgreSQL Database
* Java Spring Boot Backend
* Typescript Angular Frontend
* Docker & Docker Compose Integration
* Make utility with `Makefile`

## Requirements
* **Docker** & **Docker Compose**, may be installed via **Docker Desktop**

## QuickStart
### For Unix / WSL Users
```
# Initialise Database
make migrate

# Start the Server
make dev

# Kill the Server
make kill
```
### For Windows Users
*idk*
